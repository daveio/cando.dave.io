About the site

Can Do exists for three reasons.

Firstly, I wanted to see what the state of frontend development is like these days, as the last time I did any frontend work in earnest was quite some time ago.

Secondly, I wanted an initial frontend use-case for another project, papi, which is an extensible personal API toolbox.

Finally, I wanted to give some structure to requests people make of me, as I'm fed up with things falling through the cracks.
